Fourier Analysis
================

.. automodule:: eyetrace.signal_analysis.fourier
   :members:
   :undoc-members:
   :show-inheritance:

This submodule provides functions for frequency domain analysis, including
power spectrum estimation (FFT or Welch), band power integration, and LF/HF
ratio calculation commonly used in heart rate variability and pupillometry.
